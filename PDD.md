## Objective
Build a Progressive Web App (PWA) that allows users to capture an image of handwritten code using a device camera, convert the captured content into editable typed text, and enable users to work with that text as source code.

## Core Capabilities
- Access device camera from browser
- Capture and handle images
- Convert image content into typed text (OCR)
- Display recognized text in an editor
- Allow user to edit recognized text
- Support code-oriented workflows (JavaScript / Python)

## Tech Stack 
#### Frontend
* NEXT.js 
* React.js
* Tailwindcss
#### Backend (if needed)
* Express.js / Node.js
* MongoDB

### Hosting
